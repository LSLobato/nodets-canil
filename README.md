# nodets-canil
This repository contains the Node.js course project by b7web.

## If you are using npm:
### Global Prerequisites
`yarn add nodemon typescript ts-node`

### Instalation
`npm install`
`yarn`

### To start
`npm run start-dev`

## If you are using yarn:
### Global Prerequisites
`yarn add nodemon typescript ts-node`

### Instalation
`yarn`

### To start
`yarn start-dev`
